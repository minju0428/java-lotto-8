# java-lotto-precourse

### 💡 기능 구현 목록

#### 1. 로또 기본 기능 구현
- 금액에 따른 로또 구매 및 발행
- 로또 번호 오름차순 정렬 및 출력
- 당첨 번호 및 보너스 번호 입력

#### 2. 로직 및 모델 구현
- 로또 번호 유효성 검증 (6개, 1~45 범위, 중복)
- 당첨 등수 판별 로직 (`Rank` enum 활용)
- 총 상금 및 수익률 계산 로직 구현

#### 3. 입출력 및 예외 처리
- 구입 금액 입력 유효성 검증 (숫자 형식, 1,000원 단위) 및 재입력
- 당첨/보너스 번호 유효성 검증 (숫자 형식, 범위, 중복) 및 재입력
- 당첨 통계 출력 및 수익률 출력 (소수점 둘째 자리 반올림)

#### 4. 객체지향 설계 및 테스트
- MVC 구조 분리 (Controller, InputView, OutputView 도입)
- 프로그래밍 요구 사항 준수 (15라인 제한, Indent Depth 2, else 미사용)
- 단위 테스트 작성 (LottoTest, RankTest, StatisticsTest)
